## Git Commit 規範

| 前綴      | 用途                   |
| --------- | ---------------------- |
| feat:     | 新功能                 |
| fix:      | 修 bug                 |
| docs:     | 文件更新               |
| style:    | 格式調整（不影響邏輯） |
| refactor: | 程式重構               |
| perf:     | 效能優化               |
| test:     | 測試新增或修改         |
| build:    | 打包 / CI / config     |
| chore:    | 雜項（套件、工具更新） |
