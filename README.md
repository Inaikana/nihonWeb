# 🔖 栞 - 日文文法學習筆記

### 📖 專案簡介 / プロジェクトの概要

本專案是依據 YouTube 頻道 **「Yuka老師的日本語教室」** 的「初級文法 Basic Grammar」系列所製作的日文文法筆記網站，已徵得 Yuka 老師同意公開

本プロジェクトは、YouTubeチャンネル **「Yuka老師的日本語教室」** の「初級文法 Basic Grammar」シリーズに基づいて作成された日本語文法ノートサイトです。Yuka 先生から公開の許可をいただいております

### ✨ 核心功能與技術實作 / 核心機能と技術の実装

#### 💡 現代化開發流程 / モダンな開発プロセス

採用 React 搭配 TypeScript 開發，確保程式碼具備良好的型別定義與開發體驗。採用前後端分離架構，讓前端介面專注於呈現邏輯，後端則負責穩定的資料供應

React と TypeScript を採用し、優れた型定義と開発体験を維持しながら開発を進めています。フロントエンドとバックエンドを分離したアーキテクチャを採用し、フロントエンドは表示ロジックに、バックエンドは安定したデータ提供に専念しています

#### 🏷️ 自動化振假名渲染系統 / 自動ふりがなレンダリングシステム

針對日文平假名排版，前端透過正規表示式解析後，利用 HTML `<ruby>` 與 `<rt>` 標籤渲染，確保在響應式佈局下，平假名位置始終精確鎖定於漢字上方，不會因網頁縮放或換行導致排版錯位

フロントエンドの正規表現でパースを行い、HTML の `<ruby>`標籤と`<rt>` 標籤を用いてレンダリングすることで、レスポンシブレイアウトにおいても、平仮名の位置を常に漢字の真上に固定し、画面のズレや改行によるレイアウト崩れを防ぎます

#### 🔍 文法展示與檢索系統 / 文法表示と検索システム

系統化展示文法公式，並能透過特定類別、集數或關鍵字搜尋，快速找到對應的文法內容

文法公式を体系的に表示し、特定のカテゴリー、集数、またはキーワード検索を通じて、対応する文法内容を素早く見つけ出すことができます
